**AJAX** 实现浏览器和服务器之间通信，动态数据交互

# axios 使用

语法

1. 引入axios.js:http://cdn.jsdelivr.net/npm/axios/dist/axios.min.js

2. 使用 **axios** 函数

   * 传入**配置对象**

     ```javascript
     axios({
         url:'目标资源地址'
     }).then(result) => {
         // 对服务器返回的数据做后续处理
     }
     ```

   * 再用**.then**回调函数接受结果，并做后续处理
